**Description:**  
This project implements Huffman coding, an efficient algorithm for compressing data by creating a variable-length code for each character based on its frequency. The program allows users to input character frequencies, generates a Huffman Tree, displays character codes, and encodes and decodes messages.

## **Getting Started**
